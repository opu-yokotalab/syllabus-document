\section{機能}
本章ではシラバスシステムが持つ機能について示す。
\subsection{機能一覧}
\begin{itemize}
\item ログイン・ログアウト機能\\
ユーザのログイン・ログアウトを行う。

\item 科目一覧表示機能\\
登録されたシラバスの科目の一覧を表示する。

\item シラバス内容編集機能\\
シラバスの内容の編集を行う。

\item 新規科目の登録、削除機能\\
新規科目の登録やすでに登録されている科目の削除を行う。

\item シラバスPDF生成機能\\
新規入力、編集又は科目一覧から指定されたシラバスからPDFの生成を行い、表示する。

\item 管理者機能\\
管理者ユーザに対して全ての科目のシラバスの編集を提供する。
また教員の追加、削除を提供する。

\item パスワード変更機能\\
ログイン時のパスワードの変更を行う。

\item 所有教員の変更機能\\
科目を所有している教員の変更を行う。

\end{itemize}
